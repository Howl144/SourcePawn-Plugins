# 中文說明
用R鍵直接傳送藥丸與腎上腺素給隊友

* 原理
	* 拿著藥丸或腎上腺素－＞準心對準隊友－＞按下Ｒ鍵－＞藥丸或腎上腺素會直接給隊友
